コマンド

?mission list ミッション ロケーションの一覧を表示する。
?mission start 新しいランダム ミッションの自動発生を開始する。
?mission stop 新しいランダム ミッションの自動発生を停止する。
?mission next 次のミッションの発生タイミングを待たずに新しいランダム ミッションを発生させる。
?mission close <number> 現在進行中のミッションを番号 <number> を指定して報酬を精算して終了する。諦めるとも言う。
?mission limit-count 設定に基づいてミッションが同時に発生する数を制限するかどうかを切り替える。
?mission limit-range 設定に基づいてミッションが発生する範囲を制限するかどうかを切り替える。
?mission init <string> 名前 <string> を指定して新しいミッションを開始する。
?mission clear <number> 現在進行中のミッションを番号 <number> を指定して報酬を精算せず単に消去する。
?mission clear-all 現在進行中のミッション全てを報酬を精算せず単に消去する。
?mission prod ミッション ファウンデーションを本番モードに切り替える。
?mission debug ミッション ファウンデーションをデバッグ モードに切り替える。
?mission map mission 現在進行中のミッションの捜索円と進捗状況を地図に表示するかどうかを切り替える。
?mission map objective 現在進行中のミッションの目標物の位置を地図に表示するかどうかを切り替える。
?mission map zone ミッション ゾーンの位置を地図に表示するかどうかを切り替える。
